# v0.1 Talk Draft Prompt

Draft a 30-minute talk script from the selected idea.

Inputs:

- Selected talk
- Relevant extracted material
- Voice profile
- Humor profile
- Storytelling profile
- Structure profile
- Prior feedback

The talk should be speaker-friendly, not essay-like.

Create:

1. Title
2. Thesis
3. Audience promise
4. Slide list
5. Timing plan
6. Script by slide
7. Audience interaction moments
8. Visual or demo suggestions
9. Humor moments
10. What not to over-explain
11. Strong closing

The talk should have one central spine.

Use this rhythm:

- story before abstraction
- example before definition
- tension before lesson
- practical takeaway before philosophical conclusion

Make it sound like a real lecturer speaking to a room.
