# v0.1 Lesson Plan Draft Prompt

Draft a facilitator-ready lesson plan from the selected idea.

Inputs:

- Selected lesson-plan candidate
- Relevant extracted material
- Audience profile
- Voice profile
- Humor profile
- Storytelling profile
- Structure profile
- Teaching principles
- Prior feedback

Use `../templates/lesson-plan.md`.

The lesson must have:

1. A complete 30-minute core that stands alone.
2. A 50-minute extension that deepens practice, discussion, or application.
3. One concrete source story.
4. One worked example before abstraction.
5. One participant activity.
6. Facilitator script notes.
7. Materials or worksheet content.
8. Debrief guidance with expected answers and common wrong turns.
9. Audience, specificity, teaching, and AI-smell audits.

The lesson should make learners feel more oriented, not exposed.

Avoid turning the plan into a list of loosely related notes. One lesson gets one
teachable transformation.
