# Teaching Principles

Use these as durable quality principles, not as source imitation.

- Reduce cognitive load by introducing one main idea at a time.
- Use concrete examples before definitions.
- Prefer worked examples for unfamiliar technical material.
- Surface likely misconceptions explicitly.
- Include retrieval or prediction moments when useful.
- Match the explanation to the learner's current level.
- Use classroom exercises when students need practice, not just exposure.
