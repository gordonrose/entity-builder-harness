# Storytelling Principles

Use these as durable quality principles, not as source imitation.

- Start from a concrete incident.
- Make the tension visible before explaining the lesson.
- Keep one central spine per post or talk.
- Use contrast: expected vs actual, simple vs messy, code vs intent.
- Let specific details carry credibility.
- End with a landing that the material earns.
- Gather possible scenes before choosing the thesis. Good source work often
  contains several true lessons, but only some have enough motive, friction,
  change, and consequence to become public writing.
- Treat a hook as earned reader tension, not a clever sentence. The opening
  should make the reader want the explanation because something is already in
  motion.
- Stakes are not observations. Stakes require a cost, risk, loss,
  embarrassment, waste, false confidence, or bad decision that becomes more
  likely if the problem is ignored.
- Use the ladder of abstraction deliberately: begin with a visible, concrete
  base; climb only as far as the reader can follow; return the meaning to
  ordinary life before ending.
- When writing in first person, the narrator should serve the reader as a proxy
  or witness. Personal presence must reveal the problem more clearly, not ask
  the reader to care because the writer cares.
- Exposition has to be earned and placed. Teach background when it helps the
  reader understand a live situation, and introduce jargon slowly through plain
  language.
