# Structure Profile

## Blog Posts

Default shape for a 3-minute post:

1. Setup
2. The problem
3. The mistake, tension, or surprising detail
4. The lesson
5. The landing

## Talks

Default shape for a 30-minute talk:

- 0:00-3:00 opening
- 3:00-8:00 setup/context
- 8:00-15:00 main concept 1
- 15:00-22:00 main concept 2
- 22:00-27:00 synthesis/application
- 27:00-30:00 landing/Q&A bridge

## Teaching Assets

Prefer classroom-ready forms:

- worked examples
- debugging exercises
- spot-the-mistake prompts
- discussion questions
- whiteboard diagrams
- mini case studies
