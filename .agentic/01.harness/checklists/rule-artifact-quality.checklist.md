# Rule Artifact Quality Checklist

- Correct artifact type.
- Correct folder.
- Concise.
- Agent-usable.
- Clear `must` and `must_not`.
- Uses `source_refs`.
- Does not duplicate large guide passages.
- Does not mix task packs with layer rules.
- YAML parses.
