# Scene Card: Missing Governance Stop

## Source

`commitLogs/2026/jun/16/2026-06-16-18-59-missing-governance-stop-condition/README.md`

## What Happened

An earlier recovery used an ordinary Git stash and merge path that was
technically plausible but not governed by the harness. The author turned that
failure into a repo-wide rule: when the current workflow, gate, script, or
standard does not govern a required action, stop before acting.

## Human Pressure

The assistant can sound competent while silently substituting its own judgment
for the user's operating system.

## Visible Objects

- missing-governance standard
- AGENTS.md stop rule
- stash-based recovery example
- one-off exception log

## Why It Matters

This is the article's moral turn. AI autonomy is not only about capability; it
is about whether the system knows when it has no authority.
