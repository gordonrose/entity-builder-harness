# Article Example Ledger

Use this ledger to avoid recycling major anecdotes or examples across public
articles.

Before drafting or revising a public article, check this file. After drafting,
record the examples that did real explanatory work.

## Reuse Rules

- Do not reuse a major anecdote as the opening, main evidence, or primary
  bridge in another article.
- Do not reuse an example when a reader would feel they have already read this
  piece before.
- Minor references may repeat only when they are not doing the main
  explanatory work.
- If an example is close to one already used, gather a fresh reader-world
  example instead.
